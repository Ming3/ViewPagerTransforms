transformPage

open fun transformPage(page: View, position: Float): Unit

Apply a property transformation to the given page. For most use cases, this method should not be overridden. Instead use .transformPage to perform typical transformations.

Parameters

page - Apply the transformation to this page

position - Position of page relative to the current front-and-center position of the pager. 0 is front and center. 1 is one full page position to the right, and -1 is one page position to the left.
